val (>>=) : ('a option * 'b) -> ('b -> 'a -> 'c * 'b) -> 'c option * 'b

This operator maps a 'a option * 'b to the function f. The function f must be of type 'b -> 'a -> 'c * 'b. (x, acc) >>= f returns None, acc if x = None or (Some x', acc') if x = Some y with x', acc' = f acc y.
